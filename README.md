# Flutter Single Screen with Bottom Navigation

A Flutter project demonstrating a single screen with a functional bottom navigation bar. 
This project shows my ability to build interactive UI and structure widgets in Flutter.

## Features
- Functional Bottom Navigation Bar
- Interactive UI elements
- Clean Flutter code structure

## Tech Stack
- Flutter
- Dart

## How to Run
1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`
